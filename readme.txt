# 3. 程序会自动识别文件夹中的png jpg jpeg tif和tiff文件进行处理
# 4. 运行结束后数据自动生成在图片文件夹内的csv文件中 用excel打开
# 5. 建议SIZE的值不要设置过大 建议不超过100 否则可能因为递归过深 内存不足 导致运行失败
